## How To Use
  - git clone https://github.com/pushpender-singh-ap/Animated-Top-TabBar.git
  - cd Animated-Top-TabBar
  - npm install
  - expo start
  
## Software Used
  - VSCode
  - Expo-cli
  
## Dependencies Install
  - expo install expo-app-loading
  - expo install expo-linear-gradient
  - expo install expo-asset
